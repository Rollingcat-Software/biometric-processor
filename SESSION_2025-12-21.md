## Notes

### FIVUCSAS Architecture (Full Picture)

**6 Submodules:**
- `identity-core-api` - Spring Boot backend (Java 21)
- `biometric-processor` - FastAPI ML service (Python 3.11)
- `web-app` - React admin dashboard
- `client-apps` - Kotlin Multiplatform (Android/iOS/Desktop)
- `docs` - Documentation, SQL scripts
- `practice-and-test` - Testing environments, NFC Readers

**Shared Infrastructure:**
- PostgreSQL 16 + pgvector (`identity_core_db`)
- Redis 7 (event bus + cache)
- NGINX API Gateway

**Integration Flow:**
```
Clients → NGINX(8000) → identity-core-api(8080) ←→ biometric-processor(8000)
                              ↓                           ↓
                         PostgreSQL ←─────────────────────┘
                              ↑
                            Redis (events/cache)
```

**Key Tables in identity_core_db:**
- `biometric_data` - Face embeddings (pgvector)
- `liveness_attempts` - Liveness puzzle tracking
- `biometric_verification_logs` - Audit trail
